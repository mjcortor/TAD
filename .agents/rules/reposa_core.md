# Reposa+ Project Core Context

## 1. Identidad de Marca
- **Proyecto:** Reposa+ (E-commerce de almohadas ergonómicas).
- **Color Primario:** Índigo (#182447).
- **Estética:** Premium, calma, descanso.
- **Tipografía:** Inter / Roboto.

## 2. Requisitos Técnicos Específicos
- **Framework:** Laravel 10+.
- **Frontend:** SASS + Bootstrap 5 (Custom).
- **DB:** SQL con relaciones N:M obligatorias para Categorías y Favoritos.
